# Claude OCR Skill

**功能说明**：给 Claude Code 装上眼睛，让它能看懂 PDF 和图片里的文字、数学公式、表格和图表。

---

## 这能解决什么问题？

如果你用的是**纯文本模型**（比如 DeepSeek），Claude Code 天生是"瞎子"——它没法看书、看不懂截图、不认识公式。这个小工具就是给它配的一副眼镜。

装上之后，你把 PDF 或图片发过去，它就能：

- 把页面上的**所有文字**逐字读出来
- 把**数学公式**转成 LaTeX 格式（比如 `$$E = mc^2$$`）
- 把**表格**转成可以直接用的表格
- 把**折线图、柱状图、饼图**里的数据趋势、对比关系用文字描述出来

---

## 它怎么工作的？

整个过程是全自动的，你什么都不用管：

```
你发送一张图片或 PDF
        │
        ▼
  Claude Code 检测到文件
        │
        ▼
  自动调用 引擎1：GLM-OCR
  → 把页面上的文字、公式、表格精准提取出来
  → 数学公式自动变成 LaTeX 格式
        │
        ▼
  自动调用 引擎2：GLM-4.6V
  → 如果页面里有图表/插图，描述它们的内容
  → 比如"这是一张折线图，销量从1月到6月持续上升"
        │
        ▼
  两份结果合并输出给你
  文字/公式 100% 精准 + 图表含义清晰可读
```

整个过程你只需要说一句话，剩下的 Claude Code 自己搞定。

## 安装（推荐 ⭐）

**把下面这句话发给你的 Claude Code：**

> 帮我装这个：https://github.com/Picard-ljh/claude-ocr-skill

然后会发生什么：

1. Claude Code 自己用 git 把代码下载下来
2. 自动安装需要的依赖（Node.js 包）
3. 自动配置 CLAUDE.md（告诉以后怎么调用这个工具）
4. 检查发现缺少 API Key → **它会主动问你要**
5. 你把 Key 粘贴给它 → 它帮你设置好
6. 自动跑一个测试，确认一切正常

**就这么简单。你不需要懂代码，不需要自己敲命令。**

> 前提：你电脑上要有 Node.js。如果没有，去 https://nodejs.org 下载安装（一路点"下一步"就行）。

> API Key 去哪弄？去 https://open.bigmodel.cn 注册一个账号（免费），在控制台就能拿到。新用户有免费额度，够你用很久。

## 手动安装（备选方案）

如果上面的自动方式因为某些原因不行，你也可以自己来：

### Windows 用户

1. [下载本仓库](https://github.com/Picard-ljh/claude-ocr-skill/archive/refs/heads/main.zip) 并解压
2. 双击 `install.bat`
3. 会弹出一个黑窗口，按提示输入你的智谱 API Key
4. 完成

### Mac / Linux 用户

打开终端，输入：

```bash
cd claude-ocr-skill && bash install.sh
```

然后按提示操作。

## 常见问题

### "我不确定有没有 Node.js"

打开电脑的命令提示符（Win+R，输入 `cmd`，回车），输入 `node --version`，如果跳出来版本号（比如 `v20.10.0`）就说明有。如果没有，去 https://nodejs.org 点下载安装即可。

### "API Key 是什么？怎么拿？"

API Key 就像一把钥匙，让你能用智谱的 AI 模型。获取步骤：

1. 打开 https://open.bigmodel.cn
2. 注册一个账号（手机号就行，免费）
3. 登录后点"API Keys" → 创建一个新的 Key
4. 复制那串字符（类似 `xxxxx.xxxxxxxx`）

### "我的 Key 安全吗？"

Key 存在你电脑的环境变量里，不会被上传到任何地方。本仓库的代码里不含任何人的 Key。

### "我想换台电脑，怎么在新电脑上用？"

在新电脑上把上面那句"帮我装这个"发给 Claude Code，它会自动帮你部署好。然后用你之前在智谱拿的 Key 就行。

### "图片太大怎么办？"

GLM-4.6V 对图片大小有限制。如果遇到报错，把图片缩小一点再试。通常 1000×800 左右的图片完全没问题。

### "公式识别偶尔出错怎么办？"

这是所有 OCR 模型都有的问题，GLM-OCR 的公式准确率约 93%，目前在智谱体系里已经是最好的了。对于学术用途，建议关键公式再人工核对一下。
